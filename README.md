# Estados y Capitales de Venezuela

Una aplicación web interactiva para explorar la información de los estados y capitales de Venezuela.

## 🚀 Características

- **Búsqueda interactiva:** Encuentra estados o capitales rápidamente usando la barra de búsqueda.
- **Autocompletado:** Sugerencias inteligentes mientras escribes para facilitar la navegación.
- **Información detallada:** Visualiza la fecha de fundación y una breve descripción de cada entidad.
- **Diseño responsivo:** Adaptado para verse bien en diferentes dispositivos gracias a Bootstrap.
- **Interfaz dinámica:** Uso de animaciones suaves para mostrar y ocultar la información.

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura de la aplicación.
- **CSS3** - Estilos personalizados y diseño.
- **JavaScript (ES6+)** - Lógica de la aplicación.
- **jQuery** - Manipulación del DOM y efectos.
- **jQuery UI** - Funcionalidad de autocompletado.
- **Bootstrap 3** - Framework de diseño para componentes responsivos.
- **Normalize.css** - Consistencia en el renderizado entre navegadores.

## 📋 Requisitos

Para ejecutar este proyecto localmente, solo necesitas un navegador web moderno. 

## 🔧 Instalación y Uso

1. Clona este repositorio:
   ```bash
   git clone https://github.com/Angstromico/Estados-y-Capitales-de-Venezuela.git
   ```
2. Navega a la carpeta del proyecto:
   ```bash
   cd Estados-y-Capitales-de-Venezuela
   ```
3. Abre el archivo `index.html` en tu navegador preferido.
